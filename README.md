# my nvim setup on Mac

I love neovim

# Prerequisites

Install neovim, ripgrep(for telescope), node.
Be sure to use true color terminal like iterm2

```bash
brew install nvim
brew install ripgrep
brew install node
```

Download

```bash
cd ~/.config
git clone [git_url_here]
mv nvim-setup nvim
chmod +x setup.sh
./setup [your_name_here]
```
